Frequently Asked Questions

Everything you need to know about ngXpress.

Does ngXpress support SSR out of the box?

Yes! ngXpress comes with server-side rendering (SSR) preconfigured for SEO and performance.

What database does ngXpress use?

Prisma ORM is included by default (with SQLite), but you can easily switch to Postgres, MySQL, and more.

Is authentication included?

Yes, ngXpress ships with better-auth for secure, modern authentication flows.

Can I use Tailwind CSS?

Absolutely! Tailwind CSS 4 is fully integrated for rapid, modern UI development.
